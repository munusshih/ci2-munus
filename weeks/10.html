Description

This week, we'll review the midterm evaluation result, before we move into the rest of the semester. Then, we will review how to dynamically import content into our HTML using atomic design mindset and Javascript through a hands-on coding practice. Lastly, we will talk about Fetch and how to use asynchronous Javascript methods to build reusable HTML templates.

Assignments

The assignment is broken into two parts. Please finish the following things after collecting the data you'd like to work with.

  • Project 1-4: Component & Template

    Continue to refine your work by filling in all of the contents in using dynamic methods.

    Break your design in component and template. Think about fetch or creatElement methods while coding. You need to use Javascript to dynamically import your data. You CANNOT handcode your data into HTML.

  • Project 1-5: Presentation

    Develop a presentation for Project 2 that walks us through your process and idea. You will have 3 minutes to present, so think about how you can give us a succinct yet thorough walkthrough of your work. Make sure you cover the following points in your presentation:

    1. Content and Approach: Introduce the chosen content for your project and discuss your approach to re-designing the web experience of it.
    2. Data Collection: What does your data look like? How many are them? How are them being categorized? Think about all the lables you can give them, such as colors, sizes, weights, years made, smell, emotion and etc.
    3. Inputs & Events: Explain which HTML inputs users can use to select, filter and display the data on the website. What events are being listened? The user needs to be able to use HTML inputs to interact with the collection. It could be using it to filter out certain category, it could be using it to change numbers of cards being shown or simply using it to change the background color.
    4. Website: Show us your live site. Discuss any design decisions you made for responsiveness.
    5. Component: Show us your component and template. Discuss how your mindset changes from static to dynamic in this project.
