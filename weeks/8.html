Skew

Description

This week, we'll explore event handling in JavaScript and how to integrate it with HTML inputs to dynamically modify HTML and CSS using JavaScript. Additionally, we'll delve into creating custom functions and discuss how adhering to the DRY (Don't Repeat Yourself) principle in software design can be advantageous.

Assignments

The assignment is broken into two parts. Please finish the following things after collecting the data you'd like to work with.

  • Project 2-3: Inputs

    Start brainstorming creative interrelationships between the data collection and the HTML inputs. Consider how different pieces of information can interact with each other and how users might want to manipulate or explore this data in your website. Ewhich HTML inputs users can interact with to adjust the display or behavior of the data on the website? Does it filter out certain data? Only show certain category? How are them categorized? Through size, color, year, material or etc.

  • Project 2-4: Coded Draft

    Concentrate on crafting a preliminary version of your website using HTML/CSS/JS. Consider the various sections of your website and how they are structured layout-wise. Will there be a dedicated section for inputs? Perhaps a toggle sidebar or a dropdown menu? How will the data be displayed? Will they be within boxes, laid out in a grid or flexbox format, or presented as individual cards? Ponder these layout decisions to ensure a cohesive and visually appealing design.
