Description

This week, we'll explore the evolution of the dynamic web, examining its historical influences and key concepts such as Web 2.0, AJAX, and web applications. We'll then delve into the fundamentals of JavaScript conditionals, empowering us to make informed decisions within our code. Finally, we'll master the art of dynamically generating HTML elements from data and seamlessly integrating them into the Document Object Model (DOM), all while harnessing the power of HTML inputs to filter, select, and display our content dynamically.

Assignments

The assignment is broken into two parts. Please finish the following things after collecting the data you'd like to work with.

  • Project 2-5: Importing Data

    Finish 100% content of your web app by employing JavaScript to dynamically inject data as HTML elements into the Document Object Model (DOM). Additionally, you will utilize HTML inputs alongside conditionals to filter, select, and display specific datasets, thereby gaining insight into the intricacies of building dynamic web. Think about what component you're using for these data, and what are some unique and difference between them in terms of tagging, labeling and categoring.

  • Project 2-6: Research Libraries

    Research and list out javascript/css libraries that you are interested in using (not including jquery). Your list should include a minimum of 3 libraries, and for each library, provide the link to it, a brief description of its purpose, main features, and potential use cases. Figure out the way to create an easiest example of it. How can you use it? Why can't you use it?
