Description

This week, we will do a quick overview of web typography and how it corresponds to accessibility and CSS properties. We’ll learn about the famous "CSS Box Model" and how we can use that to control our elements. Lastly, We’ll close with a deep dive options we have for CSS layout.

Assignments

The assignment is broken into two parts.

  • Project 1-3.1: CSS Layout

    Using CSS, continue to flesh out the typography of your website. Consider the typographic system you’re developing: are you applying consistent styles to your headings, paragraphs, and any other levels of text? Start to think about your layout.

    Currently, you probably have a top-to-bottom, left-to-right reading experience. Do you want to keep that, or do something different? Consider CSS layout properties such as display, The Box Model (padding, border, margin), and positioning

  • Project 1-3.2: Typography System

    Design a simple typography guideline for your website. It should include at least h1, h2, h3 and b1, b2 (caption). Each with a description of the font size, font-weight, line height, letter spacing, and font family. You can use a tool like Figma or Illustrator to create this guideline.
