Description

This week, we will kick things off by getting to know each other and exploring the course structure. Then, we will dive into the interesting history of the internet and the web, exploring what it means for us in this class. Lastly, we'll do some fun activites on how to think like code together.

Assignments

  • Intro Survey

    In order for us to get to know you, and to get an understanding of your current skillset, please complete this survey.

  • Tech Setup

    Complete the tech setup for this class. This includes downloading Google chrome, Visual Studio Code, and Github Desktop. You also need to create an account on Github.
