Description

This week, we will explore the history of data visualization and discuss how we can utilize the mapping function to effectively visualize data within a desired range. Additionally, we will delve into the primary methods of animation in CSS—namely, transform, transition, and animation—and explore their counterparts in p5.js: lerping, frameCount, and trigonometric functions like sin() and cos(). These techniques will allow us to create dynamic and engaging visualizations using both CSS and p5.js.
